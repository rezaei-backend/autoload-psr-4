# 🚀 آموزش استفاده از این مخزن و آشنایی با استاندارد PSR-4

این مخزن یک مثال ساده و عملی برای پیاده‌سازی **استاندارد Autoloading PSR-4** در کامپوزر (Composer) است.

## 📦 این مخزن چه چیزی به من یاد می‌دهد؟

- آشنایی با فایل `composer.json` و بخش `autoload`
- نحوه استفاده از `autoload` استاندارد PSR-4 برای بارگذاری خودکار کلاس‌ها
- یادگیری ساختار پوشه‌بندی صحیح در پروژه
- تمرین عملی برای کار با Namespace در PHP

---

## 🧠 PSR-4 چیست و چرا مهم است؟

**PSR-4** مخفف **PHP Standard Recommendation 4** است. یک استاندارد رسمی از گروه FIG (PHP Framework Interop Group) که نحوه **بارگذاری خودکار (Autoloading)** کلاس‌ها را در پروژه‌های PHP مشخص می‌کند.

### 🎯 مزایای استفاده از PSR-4

| مزیت | توضیح |
| :--- | :--- |
| **عدم نیاز به require یا include** | دیگر نیازی نیست فایل‌ها را دستی وارد کنید. |
| **ساختار تمیز و استاندارد** | کد شما با فریمورک‌های بزرگ (لاراول، سیمفونی و...) هماهنگ می‌شود. |
| **خوانایی بالاتر** | Namespaceها نشان می‌دهند هر کلاس کجا قرار دارد. |
| **کارایی بهتر** | کامپوزر فقط فایل‌های مورد نیاز را بارگذاری می‌کند. |
